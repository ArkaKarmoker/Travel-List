This is a react app.<br>
To run this project locally, you can follow this step in your terminal $

1. `git clone https://github.com/ArkaKarmoker/Travel-List/`
2. `cd Travel-List`
3. `npm i`
4. `npm start`
5. `go to 'localhost:3000' in your browser`
